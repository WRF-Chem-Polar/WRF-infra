This repository contains our software infrastructure to compile and run:

 - [WPS](https://github.com/wrf-model/WPS) (the WRF pre-processing system)
 - [WRF & WRF-Chem](https://github.com/wrf-model/WRF)
 - [WRF-Chem-Polar](https://github.com/WRF-Chem-Polar/WRF-Chem-Polar), our modified version of WRF-Chem with enhancements for polar regions
